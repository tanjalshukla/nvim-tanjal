# tanjalshukla.com

My personal website built as a recreation of Neovim. It includes a file explorer,
sidebar, bufferline, line-number gutter, lualine statusline, a working `:` command-line,
and a Telescope-style fuzzy finder.

**Link: [tanjalshukla.com](https://tanjalshukla.com)**

[![The homepage rendered as a Neovim session]
(docs/screenshot.png)](https://tanjalshukla.com)

## Try the motions

You can also navigate normally, but the motions are the cool part. Maybe you can even learn Vim here!

| Keys                | Action                                              |
| ------------------- | --------------------------------------------------- |
| `j` / `k`           | scroll down / up                                    |
| `gg` / `G`          | jump to top / bottom                                |
| `]b` `[b` / `L` `H` | cycle buffers (LazyVim style)                       |
| `Space e`           | toggle the file explorer sidebar (desktop)          |
| `:`                 | command-line — try `:e now`, `:set bg=light`, `:q!` |
| `/` or `Space f f`  | fuzzy finder                                        |
| `?` or `:help`      | keymap overlay                                      |
| `Esc`               | always works                                        |

The full reference is at [/motions](https://tanjalshukla.com/motions).

## How it's built

- **[Astro](https://astro.build)**
- **One island with zero dependencies.** All interactivity (keymaps, command-line grammar,
  fuzzy finder, live statusline) is a single vanilla-TS module,
  [`src/components/islands/cmdline.ts`](src/components/islands/cmdline.ts).
  Everything else is plain HTML/CSS.
- **Tailwind v4** with a custom near-black palette defined entirely as CSS variables.
- **JetBrains Mono** theme.
- **Pixel-block wordmark** (NEOVIM-dashboard style) generated by
  [`scripts/generate-banner.py`](scripts/generate-banner.py) as an inline SVG, so its
  fills track the active theme. The OG image is drawn by
  [`scripts/generate-og.py`](scripts/generate-og.py) as a mock editor window.
- **Mobile:** vim keys don't really work well on mobile, so the bufferline doubles as a static top nav
  listing every page (plus `resume.pdf`) with zero JS; on desktop the island swaps to the nvim-style session-buffers view.

## Development

```sh
pnpm install
pnpm dev
pnpm astro check    # types
pnpm build          # static build to dist/
pnpm format         # prettier (astro plugin)
```

Regenerating assets (only needed when the wordmark, palette, or font subset changes):

```sh
python3 scripts/generate-banner.py   # src/assets/banner.svg + ascii grid
python3 scripts/generate-og.py       # public/og.png
./scripts/build-fonts.sh             # public/fonts/*.woff2 (needs fonttools)
```

Deployed on [Vercel](https://vercel.com).
